JOB PORTAL ASSIGNMENT

Referenced Document:
  -> Bootstrap Docs: https://getbootstrap.com/docs/5.3/getting-started/introduction/

Tech Stack Used:
  1) HTML
  2) Bootstrap 5
  3) Node.js
  4) Express.js
  5) MongoDB (with Mongoose)

How to Run:
1) Clone the repository or Download ZIP and extract it into your local folder:
        git clone https://github.com/YourUsername/job-Portal-assignment.git
2) Open your terminal, navigate to the project folder:
        cd job-Portal-assignment
3) Install the dependencies:
        npm install
4) Run the server:
        node server.js
5) Open your browser and navigate to:
        http://localhost:3500

 NPM Packages Used:
  1) mongoose
  2) express
  3) path
